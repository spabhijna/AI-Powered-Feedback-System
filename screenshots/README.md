## How to Take Screenshots

1. Start the server: `uvicorn app.main:app --reload`
2. Open the dashboard: `http://127.0.0.1:8000/static/index.html`
3. Take screenshots of the various views and features
4. Save them in this folder with the names listed above
5. Screenshots will automatically appear in the main README

## Recommended Tools

- **macOS**: Cmd + Shift + 4 (select area)
- **Windows**: Snipping Tool or Win + Shift + S  
- **Linux**: Screenshot tool or `gnome-screenshot -a`

## Image Specifications

- **Format**: PNG (recommended) or JPG
- **Resolution**: At least 1280x720 for clarity
- **File size**: Keep under 1MB per image if possible
- **Aspect ratio**: 16:9 or natural browser window size
